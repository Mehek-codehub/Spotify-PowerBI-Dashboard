# 🎵 Spotify Power BI Dashboard

## 📌 Project Overview

This project presents an interactive Spotify Dashboard developed using Power BI to analyze music streaming data and uncover insights into artist popularity, track performance, genres, and listener trends.

## 🎯 Objectives

* Analyze Spotify music data using interactive visualizations.
* Identify top-performing artists, albums, and tracks.
* Explore genre distribution and popularity trends.
* Provide data-driven insights through dashboards.

## 🛠️ Tools & Technologies

* **Power BI**
* **Power Query**
* **DAX (Data Analysis Expressions)**
* **Microsoft Excel / CSV Dataset**

## 📊 Dashboard Features

* Top Artists Analysis
* Most Popular Tracks
* Genre Distribution
* Album Performance
* Popularity Score Analysis
* Interactive Filters and Slicers
* Data Visualization and Reporting


## 📈 Key Insights

* Identified the most popular artists and songs based on streaming metrics.
* Analyzed trends across different genres and albums.
* Developed interactive dashboards for efficient data exploration and decision-making.



This project demonstrates skills in data analysis, data visualization, dashboard development, and business intelligence using Power BI.
